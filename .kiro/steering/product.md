# Product Overview

Voice Intelligence Desktop App - A desktop application that captures voice input, transcribes it, and enriches it through AI-powered processing.

## Core Functionality

- **Voice Pipeline**: Recording → Transcription → AI Enrichment
- **Hotkey Activation**: Quick access via keyboard shortcut
- **Processed Output**: Structured notes, formatted text, or context-aware results

## User Flow

1. User starts the desktop app
2. Activates recording via hotkey
3. Speaks their input
4. Receives AI-processed, ready-to-use output

## Key Features

- Standalone desktop application
- Voice-to-text transcription (local or API-based)
- LLM integration for content enrichment
- Direct usability of results
