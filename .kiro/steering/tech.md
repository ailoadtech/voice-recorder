# Technology Stack

## Core Framework

- **Next.js**: Primary framework for the application
- **Desktop Runtime**: Tauri

## Key Components

- **Voice-to-Text**: Flexible choice (local or API-based transcription)
- **LLM Integration**: Flexible choice for AI enrichment
- **Language**: TypeScript/JavaScript (Next.js ecosystem)

## Development Environment

- **Platform**: WSL (Windows Subsystem for Linux)
- **Project Location**: `/mnt/c/temp/_ki_project_aws/Everlast_Developer_Challenge/voice-recorder`

## Common Commands

```bash
# Development
npm run dev          # Start Next.js development server
npm run build        # Build for production
npm run start        # Start production server

# Desktop-specific (Tauri)
npm run tauri dev    # Run desktop app in development
npm run tauri build  # Build desktop app
```

## Library Guidelines

- All libraries and services are permitted
- Next.js must remain the foundation
- Choose libraries that integrate well with the desktop runtime
